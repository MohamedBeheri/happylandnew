# هابي لاند – Happy Land 🎡

نظام إدارة ملاهي **هابي لاند** – شبين الكوم / نادي الجمهورية.
نسخة تعمل **أوفلاين** بالكامل على جهازك باستخدام **XAMPP** (أو WAMP / Laragon).

---

## طريقة التشغيل خطوة بخطوة

### 1) انسخ المشروع لمجلد XAMPP
انسخ مجلد `kids` كامل إلى:
```
C:\xampp\htdocs\kids
```
(لو WAMP: `C:\wamp64\www\kids`)

### 2) شغّل XAMPP
افتح Control Panel وتأكد إن **Apache + MySQL** شغالين (أخضر).

### 3) أنشئ قاعدة البيانات
- افتح المتصفح على: `http://localhost/phpmyadmin`
- اضغط تبويب **Import** (استيراد)
- اختر الملف: `kids/database.sql`
- اضغط **Go / تنفيذ**

ده هيعمل قاعدة بيانات اسمها `kidsarea` بكل الجداول وبيانات ابتدائية.

### 4) ضع اللوجو
احفظ صورة لوجو هابي لاند باسم:
```
kids/assets/logo.png
```
(هي اللي بتظهر في صفحة الدخول والقائمة الجانبية)

### 5) افتح البرنامج
في المتصفح:
```
http://localhost/kids/login.php
```

### 6) سجّل الدخول
```
اسم المستخدم: admin
كلمة المرور:  admin123
```

## الخصائص الموجودة
- **لوحة تحكم** بإحصائيات سريعة
- **الألعاب** + **بيع التذاكر** (POS) مع رمز لكل لعبة، خصم، فواتير برقم وباركود، وطباعة حرارية
- **بيع المنتجات** (POS) مع خصم المخزون تلقائياً وفاتورة برقم وباركود
- **العملاء** (إضافة/تعديل/بحث/حذف)
- **المنتجات والمخزن** (منتجات + تصنيفات)
- **الإيرادات والمصروفات** (بيع التذاكر والمنتجات يُسجَّل فيها تلقائياً)
- **التقارير** حسب الفترة + **فاتورة فترة حرارية** بالضرائب
- **المستخدمين والصلاحيات**: أدمن (كل شيء) / كاشير (شاشته الأساسية البيع، ويشوف شاشات البيع والفواتير فقط)

## الصلاحيات
- **أدمن:** كل الصفحات (إدارة، مالية، تقارير، مستخدمين).
- **كاشير:** يدخل مباشرة على **شاشة البيع**، ويقدر يبيع تذاكر ومنتجات ويطبع الفواتير ويعيد طباعتها — من غير وصول للمالية أو التقارير أو الإعدادات.
- إنشاء حسابات الكاشير من صفحة **المستخدمين** (الأدمن فقط).

## ملاحظات
- إعدادات الاتصال في `config.php`. الافتراضي يعمل مع XAMPP مباشرة
  (المستخدم `root` بدون كلمة مرور). لو غيّرت إعدادات MySQL عدّلها هناك.
- كلمات مرور المستخدمين الجدد تُخزّن مشفّرة (bcrypt).
- مجلد `uploads/` للصور لو احتجت تضيف رفع صور لاحقاً.
